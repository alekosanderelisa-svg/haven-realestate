⚙️ Нужно подключить базу данных Firebase
Сейчас объекты видны только тебе. Чтобы клиенты видели изменения — настрой Firebase (5 минут, бесплатно):
  1. Зайди на console.firebase.google.com
  2. Создай проект → Realtime Database → Create database → выбери test mode
  3. Нажми ⚙️ Project settings → Your apps → нажми </> → дай имя → Register
  4. Скопируй данные firebaseConfig и вставь в файл firebase-config.js
  5. Сохрани файл и перезагрузи эту страницу
